About

What is this?

This is a REST API that serves Swahili slang definitions to authorized clients. See getting started section for a full guide on how on how to wire your client.

Who are you?

I am a Fullstack web and android developer with Computer Science background.

Why did you build this?

Because we I am really interested in the idea of writing an open source project and also because I can, so why not?

Technical details?

The entire project is hosted on Vercel. I used this project as an opportunity to practice pagination and generating api documentation. Got the pagination idea here btw. I used Node and MongoDB to serve the API..
